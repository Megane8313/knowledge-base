# Knowledge Base

Personal knowledge base - notes, ideas, and learning resources

## Folder Structure

```
knowledge-base/
├── notes/          # Learning notes and study materials
├── ideas/          # Project ideas and brainstorming
├── resources/      # Useful links, tools, and references
├── projects/       # Project documentation
├── templates/      # Reusable document templates
└── CLAUDE.md       # Instructions for Claude Code
```

## How to Use

- Write notes in Markdown (.md) format
- - Organize by topic in the appropriate folder
  - - Use Claude Code to search, summarize, and manage content
   
    - ## Categories
   
    - | Folder | Purpose |
    - |--------|---------|
    - | `notes/` | Tech learning, book summaries, tutorials |
    - | `ideas/` | New project ideas, feature requests |
    - | `resources/` | Bookmarks, tool lists, cheat sheets |
    - | `projects/` | Active project docs and progress |
    - | `templates/` | Reusable templates for common documents |
